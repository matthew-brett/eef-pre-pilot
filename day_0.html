Day 0¶

Morning¶

  • 10.00 - 10.45 - introduction to the course - talk slides [1].
  • 10.55 - 12.00 - Introduction to the Jupyter Notebook (see Using the Jupyter Notebooks).

2 hour break.

Afternoon¶

  • 14.00 - 14.45 - getting our data Analyzing Brexit; introducing the problem of proportions. Are there too few Brexiteers in the survey?. How would we decide?
  • 14.55 - 15.25 - another problem of proportions. If a family has 4 children, what is the probability that the family has 3 girls? A simulation.
  • 15.30 - 16.00 - solving with simulation; Conditionals, functions and loops.
[1]Source files for building the slides, and source for all the pages in this website are always available in the course Github repository.
